# 철거방법
- 현관 바닥 타일: 줄눈을 전동 드릴로 갈아낸 다음, 스크래퍼를 비스듬하게 두고 망치로 두들겨 타일을 벗겨냄.
- 장판: 걸레받이 위아래를 칼로 그은뒤, 스크래퍼를 비스듬하게 두고 망치로 두들겨 틈을 만든뒤 걷어냄.
- 천장 몰딩: 몰딩 위아래를 칼로 그은뒤, 스크래퍼를 비스듬하게 두고 망치로 두들겨 틈을 만든뒤 걷어냄.

# 기타
- 실크는 쉽게 뜯을수 있어서 도배 시공에 포함됨.
    - 합지는 뜯을수 없다고 함.
- 몰딩, 걸레받이가 짧아지면, 퍼티작업 (평탄화) 필요하고 인력 1명 추가됨.
- lxzin 시공 과정 예시
    - 걸레받이 철거 후, 마루 시공시 먼지가 많이 날리고, 걸레받이 마감이 더 수월하여 도배보다 먼저하는 경우가 많다고 함.
